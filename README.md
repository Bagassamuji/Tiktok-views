## CARA INSTALL DI TERMUX/SSH USER
```bash
pkg update -y && pkg upgrade -y
pkg install git -y
pkg install python -y
https://github.com/Bagassamuji/Tiktok-views.git
cd Tiktok-views
python Views.py atau python3 Views.py
```

```bash
apt install python3-pip
https://github.com/Bagassamuji/Tiktok-views.git
cd Tiktok-views
python3 Views.py
```
